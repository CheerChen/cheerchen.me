# AGENTS.md

## Blog Writing

When creating or editing blog posts:

1. Read `TAXONOMY.md` before choosing `categories` and `tags`.
2. Prefer existing categories and tags.
3. Add a new category only when the post clearly does not fit the existing taxonomy.
4. Tags can be more flexible, but reuse existing tags when possible.
5. For bilingual posts, keep the category and tag meanings aligned across `index.zh.md` and `index.ja.md`.
6. Do not force literal translation when a language has a more natural term.
